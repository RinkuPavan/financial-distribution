# Financial Distribution Engine (Tally Integration)

## Features
- FastAPI backend
- Tally plugin (TDL)
- XML integration
- Financial plan generation

## Setup Instructions

### Backend (Development)
- pip install -r backend/requirements.txt
- python backend/run_server.py

### EXE (Production)
- Run backend.exe

## Tally Setup
- Load financial_distribution.tdl
- Steps to load in Tally
